# QMS-Flow: System Documentation 📑

> This repository contains the official project documentation for **QMS-Flow**, an ISO 9001:2015 Quality Management System designed for Company XYZ Sdn. Bhd.

**Course:** SECJ2203 - Software Engineering  
**Team:** QualiTech Solutions

## 📌 Project Overview
QMS-Flow is a web-based platform designed to replace manual, paper-based workflows with automated digital processes to help organizations maintain ISO 9001:2015 compliance.

**Key Modules Planned:**
1. **Audit Management:** Scheduling and tracking audits with 7-year log retention.
2. **Document Control:** Centralized SOP repository with approval workflows.
3. **Customer Complaints:** Automated routing of operational issues.
4. **Corrective Actions (CAPA):** Strict data validation for resolving non-conformities.
